Rangkuman
1. Abreviation <abbr>
   Fungsi dari Abreviation adalah untuk membuat sebuah Extra Tag pada suatu kata
2. Acronym fungsinya kurang lebih sama seperti Abbr
3. Definisi <df> adalah fungsinya kurang lebih seperti Italic tetapi hanya dipakai di awal paragraf
4. Insert funsinya untuk memberi garis bawah di kata kurang lebih fungsinya sama seperti Underline 
5. Delete fungsinya adalah untuk menggaris sebuah kata
